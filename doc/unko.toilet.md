# unko.toilet

<font size=7>Text to UNKO Art!!!!</font>

## Requirements

- `toilet`
- `figlet`

### Additional installation

### apt

```sh
$ sudo apt install figlet toilet
```

### brew/linuxbrew

```sh
$ brew install figlet toilet
```

## Usage

```
unko.toilet [ -hv ] [ -w outputwidth ]
            [ --crop ] [ --flip ] [ --flop ]
            [ --180 ] [ --left ] [ --right ]
            [ --border ]
```
